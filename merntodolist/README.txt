HELLO WEB DEV TEAM!

This TodoList Coding Challenge uses a MongoDB and is written in React.

The user can:

Add a new todo (todo will be stored in database)
Delete a todo
Check any todo off as completed


To start the app you can do it one of two ways:

1. Run mongod in terminal in the merntodolist folder and npm start in the client folder.

OR

2. Run mongod in terminal in the merntodolist folder and nom run dev in the merntodolist folder (to run both the server and contents in client folder at the same time.
